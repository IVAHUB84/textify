# release-notes/

Release notes — то, что увидит пользователь продукта в changelog'е. Создаются тестировщиком (`/release`).

## Формат имени
`notes-vX.Y.Z.md`.

## Содержимое
Кратко: возможности, улучшения, исправления, изменения публичного контракта, миграционный путь, известные ограничения.

Пишутся **на русском языке**, ориентированы на пользователя продукта (не на разработчика).

Полный шаблон — в системном промпте агента `tester` (`.claude/agents/tester.md`).
