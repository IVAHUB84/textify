# Textify v0.3.1

Патч-релиз. Исправление дефекта распознавания аудио из v0.3.0.

## Исправлено
- **Аудио не распознавалось** (`services/transcribe.py`). В `model.transcribe()`
  передавались сырые `bytes`, а faster-whisper (через `av.open`) ожидает путь или
  файлоподобный объект с `read()`. На любом голосовом/аудио бот падал с
  `ValueError: File object has no read() method` и не отвечал. Байты теперь
  оборачиваются в `BytesIO`.

## Тесты
- Добавлен регрессионный тест: аргумент, передаваемый в `model.transcribe`, должен быть
  файлоподобным (не сырые `bytes`).

## Контракт
Без изменений публичного контракта (patch). OCR, аудио-приём, профиль поставки — как в v0.3.0.
