# Release Notes v1.10.2

- **Дата:** 2026-06-25
- **Тип релиза:** patch

## Кратко

Подписка на канал теперь проверяется **автоматически** — кнопку «Я подписался / Проверить»
убрали. Раньше бот держал статус подписки только в памяти и наполнял его исключительно через
нажатие этой кнопки, поэтому подписанного пользователя считал неподписанным (лимит 3 вместо 30)
и заставлял подтверждать подписку заново каждые ~10 минут. Это раздражало и было лишним.

## Что нового

### Улучшения

- **Авто-проверка подписки в `enforce_limit` (`handlers/gate.py`).** Перед тем как заблокировать
  пользователя по лимиту, бот сам вызывает `getChatMember` и, если человек подписан, тут же
  поднимает лимит до уровня подписчика — без участия пользователя. API дёргается только когда
  пользователь иначе упёрся бы в лимит (не на каждом сообщении), результат кэшируется (10 мин).
- **Убрана кнопка «Я подписался / Проверить».** Текст gate обновлён: «После подписки просто
  пришлите медиа ещё раз — подписку проверю сам, подтверждать ничего не нужно». Кнопки «Открыть
  канал» и «Пригласить друзей» остаются.

### Исправления

- Подписчик (в т.ч. создатель/админ канала) больше не упирается в free-лимит из-за того, что не
  нажал «Проверить».

## Изменения публичного контракта

- Поведение gate при исчерпании лимита: подписчик пропускается автоматически; genuine-неподписчику
  по-прежнему показывается предложение подписаться/пригласить (но без кнопки подтверждения).
- Callback-обработчик `gate:chk` сохранён для совместимости со старыми сообщениями, но новые
  gate-сообщения кнопку с ним не содержат.
- **Новые env / зависимости / миграции:** нет.
