# Release Notes v1.9.1

- **Дата:** 2026-06-25
- **Тип релиза:** patch

## Кратко

Актуализированы пользовательские тексты бота: `/start`, `/help`, а также description и
short description профиля в Telegram. Прежние тексты описывали только базовые «голос/фото → текст»
и не отражали накопившийся функционал (кнопки-действия, режимы PDF, лимиты, рефералы, работу в
группах). Изменения только в копирайте — поведение бота не меняется.

## Что нового

### Улучшения

- **`/start`** теперь перечисляет кнопки-действия (Кратко, Перевести RU↔EN, Задачи, ⏱ Тайм-коды и
  Субтитры для аудио, 📄 PDF и 📄 Скан (ч/б) для изображений), упоминает поднятие дневного лимита
  (подписка/рефералы) и работу в группах. Динамический блок со ссылкой и остатком лимита сохранён.
- **`/help`** переписан под актуальный сценарий: что делают кнопки над результатом, два режима PDF,
  распознавание в группах (ответ на медиа + упоминание бота), напоминание о лимите и совет про
  отправку изображения файлом для лучшего OCR.
- **Профиль бота** (`BOT_DESCRIPTION`, `BOT_SHORT_DESCRIPTION`) обновлён: упомянуты тайм-коды,
  субтитры, PDF и работа в группах. Лимиты Telegram соблюдены (≤512 и ≤120 символов). Применяется
  при старте через `setup_bot_profile`.

### Исправления

- Нет (релиз — актуализация текстов).

## Изменения публичного контракта

- Команды, кнопки, лимиты, OCR/транскрипция, профиль поставки — **не изменяются**. Меняются только
  тексты ответов `/start`, `/help` и метаданные профиля бота.
- **Новые env-переменные:** нет. **Новые зависимости:** нет. **Миграция:** не требуется.

## Проверка

- `ruff check .` — все проверки прошли (0 замечаний).
- `pytest` — 565 passed, 3 skipped (пропуски — pre-existing, требуют бинаря Tesseract).

  `HELP_TEXT`, проверка отсутствия строки-расшифровки лимита в `/start` без бонуса (уточнена на

- **Frontend / Playwright:** не применимо — Telegram-бот на Python.
