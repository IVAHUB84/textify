# req-textify-011: Гарантия бесплатности при росте + дискаверабилити (v1.1.0)

## Что нужно
Сделать так, чтобы рост числа пользователей **никогда не приводил к оплате** Cloudflare,
и подготовить бота к органическому привлечению. Бесплатно для пользователей и для владельца.

### 1. Дневной бюджет Cloudflare (главное)
Cloudflare Workers AI free = **10 000 «нейронов»/день** (сброс 00:00 UTC). Рост трафика
(CF Whisper + LLM-структурирование/«Кратко»/«Перевести») может выйти за лимит и начать
тарифицироваться. Нужна страховка:
- Считать обращения к CF за текущие UTC-сутки (персистентный счётчик; переиспользовать
  SQLite/каталог данных из v0.6.0; concurrency-safe UPSERT как в `services/stats.py`).
- Дневной бюджет — env `CF_DAILY_BUDGET` (консервативный дефолт с запасом под worst-case
  стоимость нейронов на вызов; конкретное значение — за архитектором).
- При достижении бюджета — **авто-деградация на локальные бесплатные движки**, без отказа
  пользователю:
  - ASR: вместо CF Whisper — локальный faster-whisper (уже есть путь);
  - LLM-структурирование: тихий фолбэк на сырой текст (уже есть);
  - действия «Кратко»/«Перевести»: понятное сообщение «дневной бесплатный лимит исчерпан,
    попробуйте завтра» (LLM-замены локально нет — деградировать корректно).
- Сброс счётчика по смене UTC-даты. Логировать переход в режим деградации (v1.0.0 logging).

### 2. Дискаверабилити (метаданные уже есть — доработать)
- `set_my_commands`/`set_my_description`/`set_my_short_description` уже задаются в
  `handlers/commands.py` (v1.0.0). Проверить и при необходимости усилить ключевые слова под
  поиск Telegram (распознавание речи, транскрипция, OCR, перевод, голос в текст, RU/EN).
- Подготовить **готовый текст листинга** для бесплатных каталогов ботов (BotoStore,
  TeleHunt и т.п.) — отдельный документ в репозитории (например `docs/listing.md`):
  название, короткое/полное описание, категории, теги. Сама подача в каталоги — ручной шаг
  владельца (зафиксировать в чек-листе деплоя).

## Зачем
Цель — максимум пользователей бесплатно для всех. Эта страховка снимает риск, что рост
(особенно режим групп и виральная петля из следующих релизов) выведет за free-tier и
заставит платить. Дискаверабилити — дешёвый органический приток.
